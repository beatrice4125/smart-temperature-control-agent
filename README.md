# Smart Temperature Control AI Agent
## 项目简介
本项目是一款面向边缘设备场景的智能环境调控AI Agent产品，结合大模型、RAG知识库与嵌入式控制能力，实现设备状态感知、用户意图理解、策略自动生成与闭环执行的全流程自动化，解决传统温控系统反应慢、误判率高、依赖人工干预的问题。

## 核心能力
1.  **用户意图理解**：支持自然语言指令交互，通过大模型解析用户需求，转化为设备可执行的控制指令。
2.  **策略智能生成**：基于环境数据、历史行为与设备状态，自动生成最优温控策略，实现能耗与舒适度的动态平衡。
3.  **RAG知识库驱动**：接入设备手册、控制规范等行业文档，让Agent具备专业知识，提升策略生成的准确性与合规性。
4.  **闭环验证与迭代**：实时监控设备执行结果，根据反馈数据自动优化策略，形成数据飞轮，持续提升控制效果。

## 技术亮点
- 使用大模型与Prompt Engineering实现自然语言交互与指令解析
- 基于RAG构建行业知识库，解决专业场景下的AI幻觉问题
- 结合嵌入式设备实现控制策略的端侧执行与数据采集
- 通过Cursor等AI开发工具快速迭代产品Demo，验证方案可行性
